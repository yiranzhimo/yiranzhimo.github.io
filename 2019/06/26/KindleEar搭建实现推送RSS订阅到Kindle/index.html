KindleEar搭建实现推送RSS订阅到Kindle

Posted on 2019-06-26 Edited on 2020-04-22 In KindleEar Valine:

文章主要介绍的是通过Google APP Engine安装KindleEar实现推送

前提条件：

科学上网
Google账号

创建 Google Cloud 项目

[https://console.cloud.google.com/](Google Cloud页面)
点击页面左上角的“选择项目”，点击弹出对话框右上角的“新建项目”，然后在“新建项目”页面中输入你喜欢的“项目名称”。
请记住项目ID。

创建 Google App Engine 应用

点击页面右上角的 [ >_ ] 图标按钮，调出云端 Shell，输入以下命令按回车：
gcloud beta app create
命令执行后会出现 Which region would you like to choose? 字样，询问选择应用的位置，在 Please enter your numeric choice: 之后输入数字 1，稍等片刻即可完成 GAE 应用的创建。

检查安装

打开浏览器输入 http://你的APPID.appspot.com 可访问你搭建的 KindleEar。
KindleEar 初始登录名和密码都是“admin”。
