Mac安装Python3设为默认版本

Posted on 2019-06-26 Edited on 2020-04-30

文章主要介绍的是在MacOS系统上安装Python3设为默认版本。

安装python3

输入命令：
brew install python3
查看python3的实际安装目录
例如：/usr/local/Cellar/python3/3.6.2

改变外部环境变量

打开配置
输入命令：
open ~/.bash_profile
写入python的外部环境变量
输入命令：
export PATH=${PATH}:/usr/local/Cellar/python3/3.6.2/bin

重命名python

输入命令：
alias python="/usr/local/Cellar/python3/3.6.2/bin/python3.6"

配置文件生效

输入命令：
source ~/.bash_profile

使用python命令查看

输入命令：
python
可以看到出来的就是python3.

查看位置

输入命令：
which python3
查看此时Python3的位置是否正确。
