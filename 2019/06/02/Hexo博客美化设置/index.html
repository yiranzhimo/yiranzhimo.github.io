Hexo博客美化设置

Posted on 2019-06-02 Edited on 2020-10-08 In Blog Valine:

文章主要叙述的就是如何调整Hexo的设置，
达到令我们满意的效果。

绑定域名

需要将GitHub的域名指定到购买的域名，
需要添加CNAME记录。

进入 blog/source 目录下，
添加并打开 CNAME 文件，
输入您的域名，重新上传您的博客，
实现域名绑定。

设置主题

以Next主题为例
在终端输入：cd blog
进行安装命令：
git clone --branch v5.1.2 https://github.com/iissnan/hexo-theme-next themes/next
这是目前的稳定版本，不是最新版。
如果想安装最新版，
输入命令：git clone https://github.com/iissnan/hexo-theme-next themes/next
之后，打开blog目录下的_config.yml，
找到theme: landscape修改为theme: next。
至于更改博客的名称、描述、头像等，
只需要修改blog/_config.yml文件
和blog/themes/next/_config.yml文件
中对应的属性名称即可
不要忘记冒号:后加空格

添加评论系统

本博客是通过Leancloud添加评论的。
首先前往：leancloud网址注册账号，
注册完以后需要创建一个应用，名字可以随便起，
然后 进入应用->设置->应用key，
获取你的appid 和 appkey 如图所示：

之后打开主题配置文件，
搜索valine，填入appid 和 appkey。

hexo常用命令

新建文章

hexo new “filename”

本地测试

hexo s

添加“阅读全文”按钮

添加 <!--more-->在任意位置。
或者修改主题配置文件
找到auto_excerpt进行修改，
设置首页文章以摘要形式显示，
length代表显示摘要的截取字符长度。

发布文章
