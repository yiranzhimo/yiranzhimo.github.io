CentOS7安装pip3、you-get和FFmpeg

Posted on 2019-06-02 Edited on 2020-04-21 In Sever Valine:

文章主要介绍的是安装python3.6.1和pip3，
同时为服务器安装FFmpeg和you-get。

安装python3.6.1

下载文件

输入命令：
wget https://www.python.org/ftp/python/3.6.1/Python-3.6.1.tgz

解压文件

输入命令：
tar -xf Python-3.6.1.tgz

进入目录

输入命令：
cd Python-3.6.1

编译安装

确保安装依赖

开始安装

逐步输入命令：

创建同步链接

输入命令：
ln -s /usr/Python-3.6.1/bin/python3 /usr/bin/python3

开始测试

python3
会出现python3的提示，
使用 quit(), exit(), 或者Ctrl-D退出命令行

安装pip3

安装动作

由于python3安装完成后默认已经带有pip3，
所以，我们使用命令创建同步连接就好了。
输入命令：
ln -s /usr/Python-3.6.1/bin/pip3 /usr/bin/pip3

重新安装

下载get-pip.py脚本

输入命令：
wget https://bootstrap.pypa.io/3.2/get-pip.py

运行脚本

输入命令：
python3 get-pip.py

python3创建pip3索引

输入命令：
ln -s /usr/python3.6.1/bin/pip /usr/bin/pip3
pip3安装完毕。

安装you-get

命令安装

输入命令：
pip3 install you-get

创建同步链接

使用命令：
find / -name you-get
找到you-get的安装路径，
之后给予同步
ln -s /usr/python3.6.1/bin/you-get /usr/bin/you-get #/usr/python3.6.1/bin/you-get替换成自己的路径

使用命令下载

常用命令如下：
you-get '视频链接' #默认情况下顶部的那个是您将获得的那个
you-get -i '视频链接' #--info/ -i选项查看所有可用的质量和格式
you-get -o /www/wwwroot/zhigengniao.xyz/Youtube -O ted '链接' #--output-dir/ -o选项设置路径/--output-filename/ -O设置下载文件的名称

安装FFmpeg

安装Nux Dextop YUM存储库

安装FFmpeg和FFmpeg开发包

输入命令：
sudo yum install ffmpeg ffmpeg-devel -y

测试安装

确认FFmpeg的安装：
ffmpeg
安装正常的话会显示：

安装完毕。
