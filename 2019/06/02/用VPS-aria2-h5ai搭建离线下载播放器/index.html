用VPS+aria2+h5ai搭建离线下载播放器

Posted on 2019-06-02 Edited on 2020-04-21 In Sever Valine:

文章主要介绍的利用一台VPS服务器，
搭建可离线下载的播放器，
通过AriaNG下载，
h5ai用作目录展示及播放。

域名

购买域名以后添加设置解析A，
添加IP地址。
不过域名有没有都okay。

安装宝塔面板

SSH连接到VPS服务器

ssh root@IP地址
之后按照提示输入密码。

命令安装

Linux面板6.9.4安装命令：
yum install -y wget && wget -O install.sh http://download.bt.cn/install/install_6.0.sh && bash install.sh
Linux面板6.9.4升级命令：
curl http://download.bt.cn/install/update6.sh|bash

安装lnmp环境

直接登录宝塔面板，按照提示安装lnmp环境，
一般情况下选择编译安装。
注意
这里有一个问题，
BT面板需要使用端口8888登录，
所以我们给IP地址开端口，
首先检查某端口是否开启：
firewall-cmd --query-port=8888/tcp --zone=public #查询8888端口是否开启,自行修改端口号
返回no即未开启，显示yes为已开启。
开启端口代码
firewall-cmd --zone=public --add-port=6800/tcp --permanent #添加80端口，如需添加其他端口，自行修改端口号
修改完成后需要重启服务器才能生效

设置网站

在BT面板主页面，
点击网站-添加站点
输入域名，设置好网站根目录，添加Mysql数据库。

安装aria2

输入代码：
wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/aria2.sh && chmod +x aria2.sh && bash aria2.sh 备用地址 wget -N --no-check-certificate https://www.moerats.com/usr/shell/Aria2/aria2.sh && chmod +x aria2.sh && bash aria2.sh
实现安装。

AriaNG管理aria2

GitHub下载地址：
https://github.com/mayswind/AriaNg/releases
下载上面的AriaNG All In One，解压之后会得到index.html，直接在本地运行即可。
最重要的一步：点击左侧“AriaNG设置”——“RPC设置”，输入在安装aria2的时候会出现配置信息，你的IP、端口和密码
AriaNG没有保存按钮，点击刷新即可。
注意
AriaNG默认的是6800的端口，
我们需要向上一步一样，开启端口。

安装h5ai

去官网下载h5ai安装包
https://larsjung.de/h5ai/
在网站根目录点击远程下载，
输入下载地址，开始下载，
下载后解压在网站目录，删除压缩包。

配置h5ai

打开宝塔面板——PHP管理，在“配置文件”中搜索“disable_functions”，将其后面的scandir、exec和passthru删除。
注意
有的函数不在的话不用管

配置Nginx

打开宝塔面板——网站——配置，在“配置文件”中“root /www/wwwroot/网站根目录”这一行后面加上下面这一行代码.
index index.html index.php /_h5ai/public/index.php;

上传AriaNG

在网站根目录，将AriaNG All In One文件夹的内容上传到服务器。
注意：AriaNG All In One的html文件不要以index.html存储，修改名称，要不然之后打开域名会直接跳转到AriaNG页面。

至此基本结束，更多设置在下一篇文章。
