Mac系统通过Wine运行Libgen Desktop

Posted on 2019-08-23 Edited on 2020-04-22 In Mac

文章主要介绍的是在MacOS系统通过Wine运行Libgen Desktop for windows。

关于

Wine

（Wine Is Not an Emulator）是一个能够在多种 POSIX-compliant 操作系统（诸如 Linux，macOS 及 BSD 等）上运行 Windows 应用的兼容层。Wine 不是像虚拟机或者模拟器一样模仿内部的 Windows 逻辑，而是將 Windows API 调用翻译成为动态的 POSIX 调用，免除了性能和其他一些行为的内存占用，让你能够干净地集合 Windows 应用到你的桌面。
我们所熟知的Crossover虚拟机安装应用就是基于Wine所实现的。

Libgen Desktop

Libgen Desktop是一个用于浏览LibGen目录本地副本的Windows应用程序。支持主要（非小说），小说和科学文章（scimag）LibGen集合。

XQuartz

XQuartz项目是一个开源工作，用于开发在OSX上运行的 X.Org X Window System 系统版本。

安装

输入命令：
brew install ruby
brew cask install xquartz
brew install wine
在Mac的如下目录，生成了wine的相关目录（ ~/.wine/drive_c ），里面最值得大家注意的是drive_c目录，这个就是我们win下面的c盘了。

运行

输入命令：
wine <your>.exe

卸载

正常卸载即可。
例如：
brew uninstall rm -r
