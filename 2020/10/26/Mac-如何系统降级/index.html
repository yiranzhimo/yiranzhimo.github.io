Mac 如何系统降级

Posted on 2020-10-26 In Mac Valine:

文章主要介绍的是 Mac 如何系统降级。

Catalina 的系统用着很不舒服，几个常用的软件都出现了问题，所以，决定降级到 Mojave，升级很简单，但是降级会麻烦一点，主要是涉及到数据的问题。

制作启动磁盘

在 App Store 搜索下载 Mac Mojave 的系统，但是一般是搜索不到的，需要从特定的入口进去

  • Mac Catalina
  • Mac Mojave
  • Mac High Sierra
  • Mac Sierra

下载好对应的包以后，会在应用商店看到 Install xxxxxx，下一步就到了真正的制作启动 U 盘的过程了
打开磁盘工具，找到外置栏的 U 盘，抹掉 U 盘，把名称设置为 xx，选择格式为 Mac OS 扩展( 日志式 )
打开终端，在终端输入命令，比如，我的是 Mojave
sudo /Applications/Install\ macOS\ Mojave.app/Contents/Resources/createinstallmedia --volume /Volumes/xx

这里的 ～/Volumes/xx 中 xx，就是之前设置的磁盘名。
⚠️ 这里要记得把查找我的 Mac退出，之前做准备功课的时候有的这里因为没有退出遇到问题。
后续就开始降级系统安装了。

降级系统安装

电脑关机，按住 option 键，再按一下电源键要一直按住 option 键，稍后屏幕上就会出现磁盘了，之后就看到了 Install Mojave，选择它，进入实用工具界面，点击磁盘工具，抹掉磁盘，随后退出，选择 安装 macOS，随后等待后续安装即可。

数据迁移

之前一直有用 Time Machine 备份数据，本来打算是直接把数据通过迁移助理迁移，但是尝试迁移 2～3 次，都中途退出了，好在所有的数据都在磁盘里面了。

这里就要强调一下备份中的重要性，电脑里面的文件，密码最好及时备份，这样电脑出了问题也不用担心，比如这次数据迁移不过去以后，应用程序都从 Time Machine 移动过来了，重要文件 iCloud 备份了，常看的书和音乐都在坚果云里面，密码加密后都在 Dropbox 里面，狡兔三窟。
