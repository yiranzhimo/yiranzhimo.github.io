通过 Hexo 和 Github 搭建博客

Posted on 2020-10-26 In Blog Valine:

文章主要介绍的是通过 Hexo 和 Github 搭建博客。

安装设置

Hexo

使用 npm 安装 Hexo，
输入：sudo npm install -g hexo-cli
这里有个小坑：Hexo 官网上的安装命令是
npm install -g hexo-cli，
安装时不要忘记前面加上 sudo，否则会因为权限问题报错。

创建目录

输入：hexo init blog创建目录
输入：cd blog进入目录
安装 npm，输入：npm install
安装完毕以后，初始化查看，
输入：hexo s
浏览器中打开网址（默认是 4000 端口）
http://localhost:4000
会看到 Hexo 的默认页面。

Github

得有一个 Github 账号，之后再进行操作。

创建仓库

登录您的 GitHub 账号，新建名称为您的用户名 xxx.github.io。
这里强调一点：
假设我的 GitHub 账号的用户名是 zzz，
那么我就应该新建名称为 zzz.github.io 的仓库。
Description 可写可不写，
勾上 Initialize this repository with a README。
点击 Create Repository 完成创建。

开启GitHub Pages

按照下面这张图片创建

修改全局配置文件

找到本地blog文件夹下_config.yml，在最后加入一下代码。
deploy: type: git repository: https://github.com/xxx/xxx.github.io.git branch: master
记得把 repository 后的所有 xxx 换成用户名。

然后，您需要为自己配置身份信息，
终端输入yourname和youremail换成您自己的 GitHub 用户名和邮箱)：
git config --global user.name "yourname" git config --global user.email "youremail"

终端cd到blog文件夹下执行生成静态页面命令：
hexo g

然后进入当前目录下，终端输入：
hexo d
完成上传之后，通过http://xxx.github.io/
( xxx 换成用户名)来访问自己刚刚上传的网站。

美化配置

将博客安装好以后，需要进行其他的一些设置。

绑定域名

需要将 GitHub 的域名指定到购买的域名，需要添加 CNAME 记录。

进入 blog/source 目录下，添加并打开 CNAME 文件，输入您的域名，重新上传您的博客，实现域名绑定。

设置主题

以 Next 主题为例
在终端输入：cd blog
进行安装命令：
git clone --branch v5.1.2 https://github.com/iissnan/hexo-theme-next themes/next
这是目前的稳定版本，不是最新版。如果想安装最新版，输入命令：
git clone https://github.com/iissnan/hexo-theme-next themes/next
之后，打开 blog 目录下的 _config.yml，找到 theme: landscape 修改为 theme: next。
至于更改博客的名称、描述、头像等，只需要修改 blog/_config.yml 文件blog/themes/next/_config.yml 文件中对应的属性名称即可，不要忘记冒号 : 后加空格。

首先前往：leancloud网址注册账号，注册完以后需要创建一个应用，名字可以随便起，然后 进入应用->设置->应用 key，获取你的appid 和 appkey。

hexo常用命令

新建文章命令：
hexo new "filename"
新建草稿命令：
hexo new draft "filename"
本地测试命令：
hexo s
添加阅读文章按钮
添加 <!--more-->在任意位置，或者修改主题配置文件，找到 auto_excerpt 进行修改，设置首页文章以摘要形式显示，length代表显示摘要的截取字符长度。

发布文章命令：
hexo clean && hexo g && hexo d
