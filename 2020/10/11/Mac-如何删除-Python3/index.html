Mac 如何删除 Python3

Posted on 2020-10-11 In Mac Valine:

文章主要介绍的是 Mac 如何系统降级。

官网安装

如果是在官网安装的话，它会安装到应用程序一个 IDLE 和 Python Launcher，Python 会安装在：
/Library/Frameworks/Python.framework
卸载步骤：

  • 第一步，卸载框架：
    sudo rm -rf /Library/Frameworks/Python.framework/Versions/x.x
  • 第二步，删除应用目录：
    sudo rm -rf "/Applications/Python x.x"
  • 第三步，删除指向 Python 的连接：
    cd /usr/local/bin/
    ls -l /usr/local/bin | grep '../Library/Frameworks/Python.framework/Versions/x.x' | awk '{print $9}' | tr -d @ | xargs rm12

    Homebrew 安装

    通过 Homebrew 安装，可以直接输入命令：
    brew install python3
    卸载命令：
    brew uninstall python3
