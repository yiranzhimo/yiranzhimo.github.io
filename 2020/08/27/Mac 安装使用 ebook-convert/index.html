Mac 安装 ebook-convert

Posted on 2020-08-27 Edited on 2020-09-06 In Mac

文章主要介绍的是 Mac 安装 ebook-convert 和简单使用。

ebook-convert 下载安装

ebook-convert 是 Calibre 的一个命令行。
下载 Calibre 以后，创建一个指向ebook-convert工具的软件链接：
sudo ln -s /Applications/calibre.app/Contents/MacOS/ebook-convert /usr/bin
但是出现问题：
ln: /usr/bin/ebook-convert: Operation not permitted
如果出现 Operation not permitted ，说明系统权限限制，需要配置环境变量的方式解决。
输入命令：
vim ~/.bash_profile
进入命令模式输入：i
这样进入了编辑模式，输入：

需要保存文件，按住 ESC，之后再命令模式，输入 :wq 保存文件即可。
刷新一下刚刚的配置:
source ~/.bash_profile
最后测试一下 ebook-convert 指令是否能正常被调用：
ebook-convert --version
弹出版本就可以，说明可以调用了。
不过当关闭了终端以后，再打开会发现提示：
zsh: command not found: ebook-convert
这时候，还是需要刷新配置：
source ~/.bash_profile

ebook-convert 使用

常用命令：
ebook-convert xxx.html xxx.mobi
会自动输出在 xxx.html 的目录
