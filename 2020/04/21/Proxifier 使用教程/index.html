Proxifier 使用教程

Posted on 2020-04-21 Edited on 2020-04-22 In Mac

文章主要介绍的是 Proxifier 的使用教程。

Proxifier

Proxifier 是一款可以针对电脑上的特定的软件设置代理的软件，会使用这个软件是因为在用一款日记软件：Journey，它需要 Google Drive 进行同步，但是这个软件奇怪在即使我在电脑上开了代理，仍然无法连接，只能通过 Proxifier 进行设置进行连接。

使用

ShadowsocksX-NG 设置

打开 Proxifier 后，在主界面，打开 Proxies，添加代理服务器，我用的是 ShadowsocksX-NG，所以代理的端口是：1086，如果用的是 Shadowsocks，代理的端口就是：1080，如果用的是 Clash X，也可以设置 HTTP 代理，选择的端口可以在 ClashX 的控制板看到。接着会弹出一个窗口，提示你是否要把这个代理服务器作为默认代理，点击 YES，然后还会有一个弹窗，提示你 ss-local 貌似进入死循环了，要不要使用建议设置自动修改，选择 OK 即可。

自定义规则

Proxifier 同时可以设置自定义规则，比如说我想要设置 Journey 走代理，可以把 Default 设置为 Direct，之后点击 Add，添加新的规则，点击 “+” 添加 Application 内的应用即可。
