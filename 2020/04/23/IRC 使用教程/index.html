IRC 使用教程

Posted on 2020-04-23 In Mac

文章主要介绍的是 IRC 的使用教程。

IRC

IRC（Internet Relay Chat 的缩写，“因特网中继聊天”）是一个位于应用层的协议。其主要用于群体聊天，但同样也可以用于个人对个人的聊天。IRC 使用的服务器端口有 6667（明文传输，如irc://irc.freenode.net）、6697（SSL加密传输，如ircs://irc.freenode.net:6697）等。
IRC采用客户机/服务器模式，它能使 Internet 用户实时地与其他用户交谈，每个用户通过客户端程序与远程主机建立连接，远程主机接受多个来自客户端程序的连接，并实现多个用户之间的实时通话。

客户端

WeeChat

它是一个轻便，快速，高可扩展的命令行基础，尤其是在 Unix，Linux，BSD，GNU Hurd，Windows 和 Mac OS 上运行的跨平台聊天客户端。

Pidgin

用于 Linux 的 Pidgin IM 客户端。

XChat

适用于 Linux 和 Windows 的 IRC 客户端，v用户能够同时连接几个聊天网络。

Textual

适用于 Mac 的 IRC 客户端。

使用过程

以 irc://irc.freenode.net 为例子
打开 Textual 后，点击 sever - add sever
输入 irc.freenode.net
可以添加服务器，服务器添加完毕后
点击 sever - add channel
输入 #freenode，即可实现添加频道。
