Moon FM 客户端同步

Posted on 2020-04-22 Edited on 2020-04-26 In Mac Valine:

文章主要介绍的是 Moon FM 不同客户端同步的使用教程。

Moon FM

Moon FM 官网
Moon FM 是一款 UI 设计精美的播客软件，支持 Mac、Windows、Linux、Android 和 IOS，真正的做到的全平台支持，但是不同的客户端之间存在同步的问题，开发者也给出了同步的解决方案，但是还是比较麻烦的，网上有一份利用 Cloudant 实现同步的方案，自己照着步骤设置完毕。

操作步骤

进入 IBM Cloud 注册；
登陆后搜索输入：Cloudant；
选择 Lite 套餐，服务名称随便取，资源组、标记都默认即可，Available authentication methods选择 Use both legacy credentials and IAM；
创建完毕后，进入到创建好的服务，点击Launch Cloudant Dashboard，进入 Cloudant 的 Dashboard 页面，创建一个数据库，记好名字：xxx；
返回创建好的服务，回到上一级页面，新建凭证，名称随意选取，角色选择 Reader，选择服务标识自动生成；
选择查看凭证，其中有：
“url”：“https://..................cloud”
而需要的地址即是：
url 地址/数据库名
https://..................cloud/xxx
填入对应的客户端即可实现同步。
