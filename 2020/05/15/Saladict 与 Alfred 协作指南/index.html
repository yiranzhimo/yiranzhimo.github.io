Saladict 与 Alfred 协作指南

Posted on 2020-05-15 In Mac

文章主要介绍的是 Saladict 与 Alfred 协作指南。

Saladict

Saladict 即沙拉查词，非常好用的查单词插件，它本身是浏览器插件，但是可以通过配置也可以在电脑上读文献时使用。

配置

插件安装

沙拉查词-聚合词典划词翻译插件安装
安装后需要在浏览器上设置全局快捷键，打开网站：chrome://extensions/shortcuts，找到沙拉查词中的在独立窗口中搜索剪贴板内容，设置快捷键（建议设置为 CMD+Shift+L ），并选择全局。

Alfred 配置

下载 Alfred workflow 脚本，安装后，选择 Hotkey 录入快捷键，快捷键随意选择即可，之后，双击 Run NSAppleScript
