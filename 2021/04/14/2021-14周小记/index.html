文章主要介绍的是 2021-14周小记。

又一周过去了，一直没有时间，就到周三才来记录一下。

Linux 把某个目录的内容复制到另一个目录中：
cp -r dir1 dir2
也可以同时复制多个文件：
cp /home/usr/dir/file{1..4} ./ xxx

Linux 查看磁盘大小：
df -h - 查看根路径的大小
du -sh xx - 查看目录大小

在 home 目录配置了一下快捷方式：
alias cd1="cd xx"

通过 Shell 命令把一个文件复制多份：
for i in {1..100};do echo -ne "hello-$i.txt ";done | xargs -n 1 cp hello.txt

统计文件行数：
less xx | wc -l

找到类似文件：
find -name xx

在这个过程中还有一个问题，source .vimrc 出现syntax命令不存在，发现原因是因为直接在终端运行了 syntax on 命令，而 shell 中不存在 syntax 这个命令，所以就会出错了，但是 vim 在运行的时候会直接读取这个文件的。

另外，vim 退出编辑模式，shift+g 会跳到文件最后一行，shift+4会跳到文件最后一行的最后一个字符。

同时，如果想删除行的话，正常模式下：
:1,10d
